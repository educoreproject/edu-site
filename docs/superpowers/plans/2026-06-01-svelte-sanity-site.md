# Svelte Sanity Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first working EDUcore/DSU public-site slice with SvelteKit, same-repo Sanity Studio, Sanity-shaped fallback content, and DSU home at `/`.

**Architecture:** SvelteKit renders the public site. Sanity Studio lives in `studio/` and owns editor-facing schemas. Frontend route loaders call content helpers that read Sanity when configured and otherwise return handoff-derived fallback content.

**Tech Stack:** Svelte `5.56.0`, SvelteKit `2.61.1`, `sv` `0.15.3`, TypeScript, Vite, Sanity Studio, `@sanity/client` `7.22.1`.

---

## File Structure

- `package.json`: root scripts for SvelteKit, Studio, checks, and builds.
- `src/app.html`: SvelteKit HTML shell.
- `src/routes/+layout.svelte`: global CSS import and layout shell.
- `src/routes/+page.ts`: DSU home loader.
- `src/routes/+page.svelte`: DSU home page.
- `src/routes/dsu/+page.ts`: re-export DSU home loader.
- `src/routes/dsu/+page.svelte`: re-export DSU home page.
- `src/lib/styles/global.css`: design tokens, fonts, responsive rules, and base styles.
- `src/lib/components/site/*.svelte`: shared chrome and primitives.
- `src/lib/content/types.ts`: normalized content contracts.
- `src/lib/content/fallback.ts`: handoff-derived DSU home and chrome content.
- `src/lib/content/queries.ts`: GROQ queries.
- `src/lib/content/site.ts`: content helper with Sanity fallback behavior.
- `src/lib/sanity/client.ts`: Sanity client configuration.
- `studio/sanity.config.ts`: same-repo Studio config.
- `studio/schemaTypes/*.ts`: schemas for DSU home, reusable objects, nav, and footer.
- `static/fonts/public-sans/*.woff2`: copied Public Sans files from the handoff.
- `static/assets/educore-logo.png`: copied approved logo asset.

```ts
import { dev } from '$app/environment';
import { sanityClient } from '$lib/sanity/client';
import { fallbackChrome, fallbackDsuHome } from './fallback';
import { chromeQuery, dsuHomeQuery } from './queries';
import type { DsuHomePage, SiteChrome } from './types';

async function fetchOrFallback<T>(query: string, fallback: T): Promise<T> {
  if (!sanityClient) return fallback;

  try {
    const result = await sanityClient.fetch<T | null>(query);
    return result ?? fallback;
  } catch (error) {
    if (dev) console.warn('Sanity read failed; using fallback content.', error);
    return fallback;
  }
}

export async function getSiteChrome(): Promise<SiteChrome> {
  return fetchOrFallback(chromeQuery, fallbackChrome);
}

export async function getDsuHomePage(): Promise<DsuHomePage> {
  return fetchOrFallback(dsuHomeQuery, fallbackDsuHome);
}
```
